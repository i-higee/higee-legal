HIGEE Privacy Policy

Effective Date: July 1, 2026

I. Preamble and Scope

To protect users' personal information and device data security, and to regulate data usage, Jiangsu Hiji New Energy Co., Ltd. ("we", "us", or "our") has formulated this HIGEE Privacy Policy ("this Policy").

This Policy applies to all users of the HIGEE software ("this APP") service and aims to clearly explain the rules for data collection, use, storage, transmission, and disclosure of this APP. By opening, installing, and using this APP, you voluntarily acknowledge and agree to all terms of this Privacy Policy.

This APP is primarily a local offline tool. It has no account registration, login, or information entry functions and does not actively collect users' personal privacy information. It only accesses the internet when the user manually triggers a version update check, solely for version verification and update package download. Apart from this, there is no background internet connection, data upload, cloud synchronization, or information push.

II. Scope of Information Collection

We strictly adhere to the principle of "minimum necessity, local use only, no active collection of private data." The scope of information collected by this APP is extremely limited and is processed locally at all times, as follows:

(I) Information Not Collected

This APP does not require registration or login. It does not collect, store, or upload any personal identity information such as user name, phone number, ID number, address, avatar, or location information. It does not collect sensitive information such as user contacts, albums, or private files.

(II) Basic Device Information Read Locally

To ensure stable local operation, adapt to device models, and conduct security risk control, this APP only reads basic device identifiers and model parameters locally on the device. All data is processed only on the user's local device.

(III) Internet Data Usage

This APP only connects to the internet briefly when the user manually triggers a version update check. It only transmits software version verification parameters to compare the latest version and download update packages. It never carries, uploads, or transmits any user device data, business data, operation records, or private information. There is no automatic internet connection, background stealth connection, or silent data upload.

III. Information Storage and Local Usage Rules

(I) Data Storage Method

All user operation data, generated data, imported data, and device operation data of this APP are stored only locally on the user's personal device. We do not set up cloud servers to store user data. There is no cloud backup, cloud retention, or data synchronization function.

(II) Data Usage Purposes

Basic device information read locally is only used for the following legitimate and necessary purposes, with no additional use:

  1. Adapting to the device system to ensure normal local operation of this APP;
  2. Conducting local security risk control to prevent software cracking, malicious tampering, and unauthorized use;
  3. Supporting version update verification to ensure users can properly upgrade the software version.

(III) Data Retention Period

All local data is retained on the user's device for a duration determined solely by the user. Users may delete all data by uninstalling the software or clearing local cache. We cannot intervene in, view, or access the user's local data.

IV. Data Transmission and Sharing Rules

(I) No Data Transmission During Normal Use

Except for manual version update verification, this APP operates entirely offline. There is no external data transmission, upload, or synchronization. There are no scenarios of background data theft or leakage.

(II) No Third-Party Sharing

We will not sell, rent, share, or transfer users' device information or local data to any third-party company, organization, or individual. We will not use user data for advertising, precision marketing, commercial promotion, or other unrelated purposes.

(III) No Third-Party Data Collection

This APP does not embed any third-party statistics, advertising, or tracking plugins. There is no scenario where third parties collect or upload user data without authorization.

V. Information Disclosure Circumstances

This APP has no mechanism for automatic data disclosure or proactive external data push. Data disclosure may only occur in the following statutory or user-initiated circumstances:

  1. When the user voluntarily exports, shares, or provides local data to third parties;
  2. When legally required to cooperate with disclosure in accordance with laws, regulations, judicial authorities, or administrative regulatory agencies' valid documents and mandatory requirements;
  3. When reasonable disclosure is necessary to protect our legitimate rights and interests, respond to legal disputes, or prevent illegal or irregular activities.

VI. User Rights

Based on the offline nature of this APP and the absence of cloud data storage, users have complete and independent control over all their local data, including:

  1. Right to Know: Users are clearly aware that this APP only reads basic device information locally, only connects to the internet for manual updates, and does not collect private data;
  2. Right to Control: Users may independently decide on software use, uninstallation, clearing of local data, and management of all software data on the device;
  3. Right to Refuse: Users may refuse manual version updates, completely blocking the only internet-connected scenario and achieving fully offline use;
  4. Right to Delete: Users may clear software data or uninstall the software at any time through device system settings, completely deleting all locally retained information.

VII. Data Security Protection

We attach great importance to user data security and have adopted multiple security protection measures tailored to the characteristics of this APP:

  1. The software operates in a fully local closed loop with no background internet channels, fundamentally eliminating the risk of data leakage;
  2. Strictly limit data access permissions, reading only the minimum necessary device parameters without obtaining unrelated device permissions or private data;
  3. Prohibit software cracking, reverse tampering, and illegal extraction of local data. Devices that violate these rules may be subject to permission restrictions to ensure local data security;
  4. No cloud data storage, avoiding security risks such as server breaches and cloud data theft.

VIII. Minor Privacy Protection

This software is open to users of all ages. Minors must use this APP with the accompaniment, knowledge, and permission of their legal guardians. We do not actively collect any personal private data of minors. All software operations and data retention are completed locally on the device. Guardians shall guide minors in the proper use of the software and manage device and local data security.

Updated policies will be published on the local page of this APP and will take effect immediately upon publication without separate notice. Your continued use of this APP constitutes acceptance of and agreement to the updated terms. If you do not accept the revised terms, you may immediately stop using the software.
